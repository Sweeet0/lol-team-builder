# LoL Team Builder (v1)

A single-page application for organizing 5v5 custom matches in League of Legends.

## Features
- **Player Management**: Register up to 14 players (7 vs 7 pool).
- **Team Assignment**: Drag-less drop-down selection for lanes.
- **Auto-Balance**: Calculates average rating and warns about imbalances.
- **Side Selection**: Randomize Blue/Red side with color-coded visuals.
- **Visuals**: Dark, premium glassmorphism aesthetics.

## How to Use
1. Clone this repository.
2. Open `index.html` in any modern browser.
3. No build steps required (Pure HTML/CSS/JS).

## Privacy
Data is currently **ephemeral** (cleared on refresh).
